## Exercise 2

**Problem:**

- Create a `Book` class with:
  - Attributes: `title`, `author`, `price`
  - Method:
    - `display()` → returns formatted string

Example:

    Input: b1 = Book("Python Basics", "Alice", 29.99)
    Output: Title: Python Basics, Author: Alice, Price: $29.99
